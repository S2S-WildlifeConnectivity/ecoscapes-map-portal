# Map Viewer Help

## Getting Started

This interactive map portal was developed by [Nature Squamish](https://naturesquamish.ca) to present, and make accessible, the mapping and modelling outputs of the [Wildlife Connectivity Project](). 

Our interactive map portal allows you to explore, and learn about, the various ecological themes, sub-themes, and indicators developed during the project, through an intuitive interface.

<hr>

## How to Use

### Step 1: Choose a Theme

Themes are high level groupings of common ecological data and serve as an entry point into the vast collection of maps we have produced.

Choosing a theme will narrow down the options available in the sub-theme dropdown.

### Step 2: Choose a Sub-theme  

Sub-themes are more specific groupings of ecological data that are related to the selected theme.

Choosing a sub-theme will narrow down the options available in the indicator dropdown.

### Step 3: Choose an Indicator

Finally, indicators are the specific data that we have mapped and modelled. It is these layers that allow you to explore and analyse the data.

Selecting an indicator will load the map and related data.

## Features

- **Interactive Maps**: View detailed environmental maps with multiple layers
- **Data Exploration**: Access comprehensive indicator data and metadata
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Theme Support**: Accessible display options with a light and dark theme.
- **URL Sharing**: Share specific views with others via the URL.

<hr>

## Info Boxes

The left panel contains three information boxes that update as you make selections:

1. **Theme Information** - Details about the selected environmental theme

2. **Sub-theme Information** - Specifics about the chosen sub-theme

3. **Indicator Information** - Comprehensive data about the selected indicator, including:
   - Description and methodology
   - Data sources and units

<hr>

## Tips & Shortcuts

- **Quick Reset**: Use the Reset Filters button to start over quickly

- **Keyboard Shortcuts**: 
  - Press `Escape` to close any open modal
  - Use `Tab` to navigate through dropdown menus

- **URL Navigation**: The URL updates automatically as you make selections, allowing you to bookmark or share specific views

- **Mobile Usage**: On mobile devices, the info panel can be toggled to give more space to the map

<hr>

## Troubleshooting

**Map not loading?**

- Ensure you've selected a theme, sub-theme, and indicator in order
- Not all indicators have an 'Analyse mode', some are only available in 'Explore mode'
- Try refreshing the page and making your selections again
- Check your internet connection

**Info panel in the way?**

- Use the Toggle Info Panel button to hide it and give more space to the map
- On mobile, the panel automatically adjusts to optimize screen space

**Can't see all options?**

- Try using the fullscreen mode for better visibility
- This portal is designed to be used on a large screen (desktop, laptop, or tablet)
